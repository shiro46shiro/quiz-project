# Quiz Project

JavaScript を用いて作成したクイズアプリです。  
コーディングやWebサイトの構造への理解を深めることを目的に学習を始めました。
問題データのスコア計算、画面遷移などを教材やAIを用いて実装しています。
AIで生成したコードは自身で読み解いた上で使用し学習しています。
後半では AWS を利用したユーザー毎のスコア保存とランキング機能を作成したいと考えています。

---

## 現在の機能
- スタート画面 (index)
- クイズの出題 (app)
- 正誤判定とスコア計算 (app)
- 結果画面の表示(result)

---

## 今後実装予定の機能
- ユーザーごとのスコア保存
- スコアランキング機能
- クイズを増量してランダム15問出題
- 問題データの JSON 化

- HTML / CSS / JavaScript
